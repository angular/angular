Deferrable Views (Təxirə salına bilən görünüşlər)
Deferrable views (Təxirə salına bilən görünüşlər — həmçinin @defer blokları kimi tanınır) səhifənin ilkin render edilməsi (initial rendering) üçün birbaşa vacib olmayan kodların yüklənməsini təxirə salaraq, tətbiqinizin ilkin paket ölçüsünü (initial bundle size) azaldır.
