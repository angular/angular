<docs-decorative-header title="Что такое Angular?" imgSrc="adev/src/assets/images/globe.svg" gradientBackground="true">
  Angular — веб-фреймворк, который помогает<br/>
  разработчикам создавать быстрые и надёжные приложения, которые нравятся пользователям.
</docs-decorative-header>

<div style="margin: 2em">
  Поддерживаемый специальной командой в Google, Angular предоставляет широкий набор инструментов, API и
  библиотек, упрощающих и оптимизирующих рабочий процесс разработки. Angular даёт
  надёжную платформу для создания быстрых и стабильных приложений, которые масштабируются вместе с ростом
  вашей команды и кодовой базы. Angular.dev — официальный дом документации Angular.
</div>

<docs-card-container headerTitle="Разрабатывайте приложения быстрее, чем когда-либо" headerImgSrc="adev/src/assets/images/browser-bolt.svg">
  <docs-card title="CLI" href="tools/cli" link="Angular CLI" iconImgSrc="adev/src/assets/icons/cli.svg">
    Angular CLI запускает ваш проект менее чем за минуту с командами, необходимыми для роста до развёрнутого production-приложения.
  </docs-card>
  <docs-card title="DevTools" href="tools/devtools" link="Angular DevTools" iconImgSrc="adev/src/assets/icons/dev-tools.svg">
    Angular DevTools работает рядом с инструментами разработчика браузера. Помогает отлаживать и анализировать приложение, включая инспектор дерева компонентов, представление дерева внедрения зависимостей и настраиваемую flame chart профилирования производительности.
  </docs-card>
  <docs-card title="ng update" href="update" link="ng update" iconImgSrc="adev/src/assets/icons/ng-update.svg">
    `ng update` в Angular CLI выполняет автоматические преобразования кода, которые автоматически обрабатывают рутинные breaking changes, значительно упрощая обновление до новых major-версий. Поддержание актуальной версии помогает сохранять приложение максимально быстрым и безопасным.
  </docs-card>
  <docs-card title="Language Service" href="tools/language-service" link="Language Service" iconImgSrc="adev/src/assets/icons/language-service.svg">
    IDE Language Service Angular обеспечивает автодополнение кода, навигацию, рефакторинг и диагностику в реальном времени в вашем любимом редакторе.
  </docs-card>
</docs-card-container>

    <p>Angular ориентирован на стабильность для некоторых крупнейших продуктов Google, включая Google Cloud. Это обеспечивает тщательное тестирование изменений, обратную совместимость и инструменты миграции, когда это возможно.</p>
  </docs-card>

Окна Long Term Support (LTS) гарантируют получение критических исправлений безопасности, когда они нужны. Встроенные инструменты обновления, руководства и автоматические схемы миграции помогают поддерживать приложения в актуальном состоянии с учётом последних достижений фреймворка и веб-платформы.
  </docs-card>
</docs-card-container>
