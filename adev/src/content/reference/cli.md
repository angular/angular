# Справочник CLI {#cli-reference}

| Команда                            | Псевдоним  | Описание                                                                                                                  |
| :--------------------------------- | :--------- | :------------------------------------------------------------------------------------------------------------------------ |
| [`add`](cli/add)                   |            | Добавляет поддержку внешней библиотеки в ваш проект.                                                                      |
| [`analytics`](cli/analytics)       |            | Настраивает сбор метрик использования Angular CLI.                                                                        |
| [`build`](cli/build)               | `b`        | Компилирует приложение или библиотеку Angular в выходную директорию dist/ по указанному пути.                             |
| [`cache`](cli/cache)               |            | Настраивает постоянный дисковый кэш и отображает статистику кэша.                                                         |
| [`completion`](cli/completion)     |            | Настраивает автодополнение Angular CLI для вашего терминала.                                                              |
| [`config`](cli/config)             |            | Получает или задаёт значения конфигурации Angular в файле angular.json рабочего пространства.                             |
| [`deploy`](cli/deploy)             |            | Вызывает builder развёртывания для указанного проекта или проекта по умолчанию в рабочем пространстве.                    |
| [`e2e`](cli/e2e)                   | `e`        | Собирает и запускает приложение Angular, затем выполняет end-to-end тесты.                                                |
| [`extract-i18n`](cli/extract-i18n) |            | Извлекает i18n-сообщения из исходного кода.                                                                               |
| [`generate`](cli/generate)         | `g`        | Генерирует и/или изменяет файлы на основе схематика.                                                                      |
| [`lint`](cli/lint)                 |            | Запускает инструменты линтинга для кода Angular-приложения в указанной папке проекта.                                     |
| [`new`](cli/new)                   | `n`        | Создаёт новое рабочее пространство Angular.                                                                               |
| [`run`](cli/run)                   |            | Запускает цель Architect с опциональной пользовательской конфигурацией builder, определённой в вашем проекте.             |
| [`serve`](cli/serve)               | `s`, `dev` | Собирает и запускает ваше приложение с пересборкой при изменении файлов.                                                  |
| [`test`](cli/test)                 | `t`        | Запускает unit-тесты в проекте.                                                                                           |
| [`update`](cli/update)             |            | Обновляет ваше рабочее пространство и его зависимости. См. [руководство по обновлению](/update-guide).                    |
| [`version`](cli/version)           | `v`        | Выводит версию Angular CLI.                                                                                               |
