# Отсутствует значение Control

Эта ошибка возникает, когда вы вызываете `setValue` на `FormGroup` или `FormArray`, но переданное значение не содержит запись для одного из зарегистрированных controls.

`setValue` строг — он ожидает значение для каждого control. Если нужно обновить только некоторые controls, используйте вместо этого `patchValue`.

## Отладка ошибки {#debugging-the-error}

Проверьте, какой control указан в сообщении об ошибке, затем убедитесь, что объект значения его включает.

Частый источник этой ошибки — spread объекта, у которого нет всех ключей:

```typescript
const someValue = {first: 'Nancy'}; // 'last' is missing

form.setValue({...someValue}); // throws NG01002
```

Это может произойти, если `someValue` приходит из ответа API, частичного обновления состояния или типа, который не полностью соответствует структуре формы. В таких случаях либо явно заполните отсутствующие ключи, либо переключитесь на `patchValue`.

### FormGroup {#formgroup}

```typescript
const form = new FormGroup({
  first: new FormControl(''),
  last: new FormControl(''),
});

// 'last' is not in the value — throws NG01002
form.setValue({first: 'Nancy'});

// both controls are covered — works fine
form.setValue({first: 'Nancy', last: 'Drew'});
```

### FormArray {#formarray}

```typescript
const formArray = new FormArray([new FormControl(''), new FormControl('')]);

// only one value for two controls — throws NG01002
formArray.setValue(['Nancy']);

// one value per control — works fine
formArray.setValue(['Nancy', 'Drew']);
```
