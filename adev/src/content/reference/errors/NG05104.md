# Root element was not found {#root-element-was-not-found}

Компоненты начальной загрузки определяются в свойстве `bootstrap` декоратора `@NgModule` или в качестве первого параметра `bootstrapApplication` для Standalone-компонентов.

Эта ошибка возникает, когда Angular пытается загрузить один из этих компонентов, но не может найти соответствующий узел в DOM.

## Отладка ошибки {#debugging-the-error}

Проблема возникает при несоответствии селектора и тега

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class App {}
```

<!-- prettier-ignore -->
```html
<html>
  <app-root></app-root> <!-- Не соответствует селектору -->
</html>
```

Замените тег на правильный:

<!-- prettier-ignore -->
```html
<html>
  <my-app></my-app> <!-- OK -->
</html>
```
