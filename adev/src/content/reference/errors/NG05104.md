# Корневой элемент не найден

Загружаемые компоненты определяются в свойстве `bootstrap` декоратора `@NgModule` или в качестве первого параметра
`bootstrapApplication` для standalone-компонентов.

Эта ошибка возникает, когда Angular пытается загрузить один из этих компонентов, но не может найти соответствующий
узел в DOM.

## Отладка ошибки {#debugging-the-error}

Проблема возникает, когда селектор не совпадает с тегом:

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class App {}
```

<!-- prettier-ignore -->
```html
<html>
  <app-root></app-root> <!-- Не совпадает с селектором -->
</html>
```

Замените тег на правильный:

<!-- prettier-ignore -->
```html
<html>
  <my-app></my-app> <!-- OK -->
</html>
```
