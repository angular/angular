# Миграция на синтаксис потока управления

[Синтаксис потока управления](guide/templates/control-flow) доступен начиная с Angular v17. Новый синтаксис встроен в
шаблон, поэтому вам больше не нужно импортировать `CommonModule`.

Этот схематик переводит весь существующий код вашего приложения на использование нового синтаксиса потока управления.

Запустите схематик с помощью следующей команды:

```shell
ng generate @angular/core:control-flow
```

## Критические изменения {#breaking-changes}

### Повторное использование представления в `@for` {#for-view-reuse}

При использовании блока `@for`, если свойство в выражении `track` изменяется, но ссылка на объект остаётся прежней (изменение на месте), Angular обновляет привязки представления (включая inputs компонентов) вместо того, чтобы уничтожать и пересоздавать элемент.

Это отличается от поведения `*ngFor`, который в аналогичном сценарии выполнял перемонтирование (уничтожение и пересоздание) элемента, если функция `trackBy` возвращала другое значение.
