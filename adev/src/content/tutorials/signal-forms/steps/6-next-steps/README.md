# Поздравляем! {#congratulations}

Вы завершили руководство по Signal Forms и создали полноценную форму входа с нуля!

## Чему вы научились {#what-you-learned}

В ходе этого руководства вы изучили основы Angular Signal Forms:

1. **Модели формы** — Создание типобезопасных данных формы с помощью сигналов и функции `form()`
2. **Привязка полей** — Использование директивы `[formField]` для двусторонней привязки данных и отображения поля с помощью `value()`
3. **Валидация** — Применение встроенных валидаторов (таких как `required()`, `email()`) с пользовательскими сообщениями
4. **Отображение ошибок** — Условное отображение ошибок валидации на основе состояния поля
5. **Отправка формы** — Обработка отправки формы с помощью функции `submit()`
